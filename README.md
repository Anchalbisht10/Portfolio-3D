## 🚀 Deployment

This project is deployed using **Vercel** and can be easily hosted on any modern hosting platform.

---

## ⚙️ Installation & Setup

---

Clone the repository:[https://github.com/Anchalbisht10/Portfolio-3D]
---
Install dependencies:
npm install --legacy-peer-deps

Run development server:
npm run dev

Build for production:
npm run build
